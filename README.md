# Chris Shen Portfolio

Vue + Vite portfolio site with bilingual content powered by `vue-i18n`.

## Local Development

```bash
npm install
npm run dev
```

## Deployment

The site is configured for GitHub Pages through `.github/workflows/deploy.yml`.
After pushing to `main`, GitHub Actions builds the Vue app and deploys `dist`.

Expected public URL:

```text
https://chrisshen1106.github.io/my-Web/
```

In the repository settings, set **Pages** source to **GitHub Actions**.
